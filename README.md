# 🏏 About the Project

Cricket Score is a modern and intuitive web application designed to provide a seamless cricket scoring experience for local tournaments, clubs, schools, academies, and friendly matches. Built with **React.js**, the application offers real-time score updates, comprehensive player statistics, automated match calculations, and an easy-to-use interface for scorers.

Whether it's a practice match or a competitive tournament, Cricket Score makes scoring faster, more accurate, and completely digital.

- Real-time score updates

# ⚙️ Installation

Clone the repository

```bash
git clone https://github.com/yourusername/cricket-score.git
```

Navigate to the project

```bash
cd cricket-score
```

Install dependencies

```bash
npm install
```

Run the development server

```bash
npm run dev
```

Build for production

```bash
npm run build
```
